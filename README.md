A modern ReactJS application for creating responsive typography systems using CSS `clamp()`, viewport units (`vw`), and modular scale ratios. The playground enables designers and developers to visualize, customize, and export fluid typography values in real-time without relying on complex media queries.

## Project Description

Typography is one of the most important aspects of responsive web design. Traditional approaches often require multiple media queries to manage text sizes across different devices.

The **Typography Typeface Fluid-Scale Playground** solves this problem by generating fluid typography using modern CSS mathematical functions. Users can configure viewport boundaries, base font sizes, typography scale ratios, colors, and fonts while instantly previewing the results across multiple screen sizes.

The application provides an interactive environment for experimenting with typography systems while generating production-ready CSS for real-world projects.

## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/fluid-scale-playground.git
```

### 2. Navigate to the Project Directory

```bash
cd fluid-scale-playground
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start the Development Server

```bash
npm run dev
```

### 5. Build for Production

```bash
npm run build
```

### 6. Preview Production Build

```bash
npm run preview
```
